# Corne Graphite Layout (3x6 + 3)

This keymap is designed for Corne (`crkbd`) with:
- Graphite alpha placement on base layer
- Simple, VIA-friendly layers (`MO` only)
- No tap dance, combos, key overrides, or other non-VIA extras

## Layers
- `0`: Base (Graphite + simple punctuation)
- `1`: Symbols and numbers
- `2`: Navigation

## Build

```sh
qmk compile -kb crkbd -km graphite_corne
```
